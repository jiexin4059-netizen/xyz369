# 公司账目管理系统

一个简单易用的公司账目管理网页，适合编程新手学习和使用。直接用浏览器打开即可，无需安装其他软件。

## 功能

### 登入与公司

- **统一登入账号**：一个登入账号 + 密码，可访问多家公司
- **切换公司**：登入后在顶部下拉框切换公司，无需重新输入密码
- **多公司数据隔离**：每家公司的银行和账目互不影响

### 单据录入

每条记录包含：

| 字段 | 说明 |
|------|------|
| 时间 | 交易发生时间 |
| Ref | 参考号 / 单据编号 |
| In | 收入金额 |
| Out | 支出金额 |

### 系统设置（管理员）

- **添加员工**：为当前公司新增员工号和姓名
- **添加银行**：为当前公司新增银行账户
- **创建登入账号**：为新员工设置登入账号和密码

### 导出 Excel

- **导出本银行 Excel**：导出当前银行的所有单据（CSV 格式，Excel 可直接打开）
- **导出本公司全部 Excel**：导出当前公司所有银行的单据

## 使用方法

1. 用浏览器打开 `index.html`
2. 输入登入账号和密码
3. 若可访问多家公司，选择要进入的公司
4. 选择银行，填写时间、Ref、In、Out，点击「添加单据」
5. 需要时点击「导出 Excel」下载报表

## 数据结构说明

系统为后续扩展预留了清晰结构：

```
登入账号 → 可访问多家公司 → 每家公司有独立银行和员工 → 账目按「公司 + 员工 + 银行」分开保存
```

同一登入账号访问不同公司时，员工号可以相同（如 admin 在两家公司都是 EMP001），数据仍然分开。

## 注意事项

- 数据保存在浏览器本地，换电脑或清除缓存会丢失
- 导出文件为 CSV 格式，用 Excel 打开即可
- 如需多人实时共享、云端备份，后续可升级为服务器 + 数据库版本
