# 公司账目管理系统

这是一个给编程新手也能直接使用的静态网页版本公司账目管理系统。

## 如何使用

1. 直接用浏览器打开 `index.html`。
2. 输入员工号登入，例如 `EMP001`。每个员工号会看到自己的账目记录。
3. 登入后会看到 3 个银行页面：
   - MBB XX
   - PBB XX
   - CIMB XX
4. 进入其中一个银行页面后填写：
   - 收款时间
   - Ref 单据编号
   - In（收）
   - Out（出）
5. 点击“添加记录”保存。

日期会自动记录为当天日期，不需要自己输入。

数据会保存在当前浏览器的 localStorage 中，不需要数据库。
