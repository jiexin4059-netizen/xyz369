# 公司账目管理系统

一个简单易用的公司账目管理网页，适合编程新手学习和使用。直接用浏览器打开即可，无需安装其他软件。

## 功能

- **员工登入**：每个员工有独立的员工号和密码
- **多银行管理**：登入后可选择不同银行（中国银行、工商银行、建设银行等）
- **单据录入**：每条记录包含以下字段
  - **时间**：交易发生时间
  - **Ref**：参考号 / 单据编号
  - **In**：收入金额
  - **Out**：支出金额
- **自动汇总**：显示当前银行的总收入、总支出和结余
- **数据保存**：记录自动保存在浏览器本地，刷新页面不会丢失

## 使用方法

1. 用浏览器打开 `index.html` 文件（双击或在浏览器中选择「打开文件」）
2. 使用员工号和密码登入
3. 点击上方银行标签，选择要录入的银行
4. 填写时间、Ref、In、Out，点击「添加单据」
5. 下方表格会显示所有记录，并可查看汇总数据

每个员工的数据互相独立，同一员工在不同银行的数据也分开保存。

## 注意事项

- 数据保存在当前浏览器的本地存储中，换电脑或换浏览器需要重新录入
- 清除浏览器缓存会导致数据丢失
- 如需多人共享数据或更高安全性，后续可升级为带数据库的服务器版本
